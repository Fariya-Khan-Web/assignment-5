Important FAQ

What is the Document Object Model (DOM)?

DOM(Document Object Model) is a programming interface where the HTML document converts into object creating a tree-like structure.

How do you select an element from the DOM?
To select an element from DOM, you can use these 5 methods-

1.document.getElementById()

2.document.getElementsByClass()

3.document.getElementsByTagName()

4.document.quarrySelector()

5.document.quarrySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript, where event listener is added directly in the parentnode instead of chindnode.
Event delegation technique increases code readability, simplifies management, reduces the number of code lines.

How do you manipulate an element's attributes and styles using the DOM?
An elements attributes can be manipulated by '.setAttribute()' , '.setAttribute()' and '.removeAttribute()'.

To manipulate styles by DOM, '.style' is used.
